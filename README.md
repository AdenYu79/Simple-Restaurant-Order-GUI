# Restaurant Order GUI

## Project Description
Restaurant Order GUI is a Java-based restaurant ordering system that allows customers to place orders, track order progress, and process payments through a graphical user interface. The system simulates a real-world restaurant workflow including customer ordering, kitchen preparation, payment handling, and order status updates.

## Design Patterns Used
- **Factory Pattern** – creates menu items
- **Strategy Pattern** – supports different payment methods
- **Observer Pattern** – updates users when order status changes
- **Iterator Pattern** – browses menu items and active orders without exposing internal structure

## Technologies
- Java
- JavaFX

## Main Features
- Browse restaurant menu
- Create customer orders
- Process payments
- Track order status from preparing to ready
- View menu items and active orders
